## Project 1 — To-Do List App

A simple command-line **To-Do List** application that helps users manage their daily tasks efficiently.

### Features
- ✅ Add new tasks
- 📋 View all current tasks
- ❌ Delete tasks by name
- ✏️ Update/rename existing tasks
- 🔁 Continuous menu loop until user exits

### File
`todo_list.py`

## Project 2 — Expense Tracker App

A command-line **Expense Tracker** that lets users record, view, update, and delete expenses across predefined categories.

### Features
- ➕ Add expenses under categories (Food, Travel, Shopping, Bills)
- 👁️ View all recorded expenses
- 🗑️ Delete expenses by category
- 🔄 Update expense amounts
- 💰 Calculate total expenses
- ⚠️ Input validation with error handling

### File
`expense_tracker.py`

## Project 3 — Random Password Generator

A lightweight **Password Generator** that creates strong, random passwords of a user-specified length.

### Features
- 🔐 Generates passwords using letters, digits, and special characters
- 📏 Accepts custom length between 8 and 64 characters
- ✅ Input validation for non-numeric and out-of-range entries

### File
`password_generator.py`

Make sure you have **Python 3** installed. Then run any project using:

```bash

```
```bash

```
```bash

```

No external libraries are required — all projects use Python's built-in modules only.
